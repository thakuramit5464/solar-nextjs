# Solar Next.js Starter

Minimal Next.js (App Router) + TypeScript + Tailwind starter for a solar installation website.

Core pages included:

- Home
- Services
- Subsidy (India-focused content starter)
- Calculator (simple estimate)
- About
- Contact (form stub)

Tech:

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Next SEO

Quick start:

```bash
cd solar-nextjs
npm install
npm run dev
```

Notes / next steps:

- Replace placeholder contact form with API route or integration (SendGrid, Zapier, Firebase Functions).
